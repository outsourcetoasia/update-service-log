Log your Updates and show it in the Dashboard!

== Description ==

WordPress core, themes and plugins updates will be automaticly recorded and logged.

##How it works

The Update Service Log plugin is very easy to use; simply install and activate it on your WordPress website and it will automatically capture all the updates you make.

== Installation ==

=== Manually ===

1. Download the plugin from the [Github repository](https://github.com/outsourcetoasia/update-service-log/releases)
1. Unzip the zip file and upload the `update-service-log` folder to the `/wp-content/plugins/` directory
1. Activate the Update Service Log plugin through the 'Plugins' menu in WordPress
